# 🚀 KerjaRia

KerjaRia adalah platform berbasis website yang menghubungkan siswa dan lulusan SMA/SMK dengan UMKM melalui sistem pelatihan keterampilan berbasis progresi level dan praktik kerja sederhana (micro-work) untuk membangun portofolio serta meningkatkan kesiapan menghadapi dunia kerja.

Konsep utama website ini adalah menggabungkan proses belajar dan pengalaman kerja dalam satu platform. Pengguna tidak langsung mendapatkan akses pekerjaan, tetapi harus menyelesaikan pelatihan dan tugas terlebih dahulu untuk meningkatkan level kemampuan mereka.

Setelah mencapai level tertentu dan melalui proses verifikasi akun, pengguna dapat mengakses fitur micro-work dari UMKM. Dengan sistem tersebut, pengguna dapat memperoleh pengalaman kerja sederhana secara bertahap sebelum memasuki dunia kerja yang sebenarnya.

Selain membantu pengguna, website ini juga membantu UMKM memperoleh tenaga kerja pemula untuk membantu pekerjaan digital dengan biaya yang lebih terjangkau dan sistem yang lebih terstruktur.

## ✨ Fitur Utama

- **Sistem Level & Progresi**: Kurikulum terstruktur yang dibagi menjadi beberapa misi. Pengguna naik level dengan mengumpulkan XP (Maksimal Level 30).
- **Misi Pembelajaran**: Modul pembelajaran interaktif (Materi, Quiz, Assignment) yang berfokus pada berbagai bidang (Frontend, Backend, UI/UX, Data Analyst, dll).
- **Micro-Work (Pekerjaan Mini)**: Akses ke pekerjaan dari UMKM yang terbuka hanya untuk pengguna yang telah memenuhi level/kemampuan tertentu.
- **Multi-Step Onboarding**: Alur registrasi interaktif untuk mengetahui minat dan profil pengguna.
- **Dashboard Interaktif**: Antarmuka dashboard bergaya gamifikasi yang rapi, modern, dan sangat interaktif.

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun menggunakan teknologi modern web development:
- **[React 19](https://react.dev/)**: Library utama untuk membangun antarmuka pengguna.
- **[Vite](https://vitejs.dev/)**: Build tool yang sangat cepat untuk environment development.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utility-first untuk styling.
- **[Framer Motion](https://www.framer.com/motion/)**: Library animasi untuk React (digunakan di landing page).

## 🚀 Cara Menjalankan Proyek Secara Lokal

Ikuti langkah-langkah berikut untuk menjalankan KerjaRia di komputermu:

### Prasyarat
Pastikan kamu sudah menginstal **Node.js** di komputermu.

### 1. Kloning Repositori & Masuk ke Direktori
Jika menggunakan Git, clone repositori ini. Jika sudah, buka terminal dan masuk ke folder proyek:
```bash
cd "SEEFEST - 26"
```

### 2. Instal Dependensi
Jalankan perintah berikut untuk mengunduh semua package yang dibutuhkan:
```bash
npm install
```

Pastikan juga dependensi Tailwind CSS sudah terinstal:
```bash
npm install @tailwindcss/vite tailwindcss
```

### 3. Jalankan Development Server
Mulai server lokal Vite dengan perintah:
```bash
npm run dev
```

### 4. Buka di Browser
Setelah server berjalan, kamu bisa membuka alamat berikut di browsermu:
```text
http://localhost:5173/
```
*(atau alamat lain yang muncul di terminalmu)*
